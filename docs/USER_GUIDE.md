## Getting Started

### Accessing the Platform

1. Open your web browser (Chrome, Firefox, Safari, or Edge)
2. Navigate to the platform URL
3. You'll see the **Role Selection** page

### Choosing Your Role

On the landing page, you'll see two options:

| Role                            | Best For                                      | Actions                                  |
| ------------------------------- | --------------------------------------------- | ---------------------------------------- |
| 📣 **I Need Help (Report)**     | Victims, witnesses, anyone needing assistance | Report emergencies, find camps, get help |
| 🤝 **I Want to Help (Respond)** | Volunteers, NGOs, donors                      | View reports, volunteer, donate          |

> 💡 **Tip:** You can switch between roles anytime by clicking the home button or navigating to the main page.

## For Reporters

As a reporter, you can submit emergency reports to alert responders and authorities.

### 1. Report a Missing Person

**Purpose:** Report someone who is missing during or after a disaster.

**Steps:**

1. Click **"I Need Help"** from the landing page
2. Select **"Missing Persons"** from the menu
3. Click **"Report Missing Person"**
4. Fill in the form:

| Field              | Required    | Description                     |
| ------------------ | ----------- | ------------------------------- |
| Full Name          | ✅ Yes      | Complete name of missing person |
| Age                | ✅ Yes      | Age in years                    |
| Gender             | ✅ Yes      | Select from dropdown            |
| Description        | ✅ Yes      | Physical appearance, clothing   |
| Last Seen Location | ✅ Yes      | Click on map or enter address   |
| Last Seen Date     | ✅ Yes      | When they were last seen        |
| Your Name          | ✅ Yes      | Reporter's name                 |
| Contact Number     | ✅ Yes      | Phone for updates               |
| Photo              | ❌ Optional | Upload a clear photo            |
| Additional Info    | ❌ Optional | Any other relevant details      |

5. Click **"Submit Report"**
6. You'll receive a confirmation with a reference number

> ⚠️ **Important:** Provide accurate contact information so responders can reach you.

### 2. Report a Disaster

**Purpose:** Report a disaster incident (flood, landslide, fire, etc.)

**Steps:**

1. Navigate to **"Disaster Reports"**
2. Click **"Report Disaster"**
3. Fill in the form:

| Field           | Required    | Description                      |
| --------------- | ----------- | -------------------------------- |
| Disaster Type   | ✅ Yes      | Flood, Landslide, Fire, etc.     |
| Severity        | ✅ Yes      | Low, Medium, High, Critical      |
| Description     | ✅ Yes      | What happened, current situation |
| Location        | ✅ Yes      | Click on map or enter address    |
| People Affected | ❌ Optional | Estimated number                 |
| Casualties      | ❌ Optional | If known                         |
| Immediate Needs | ❌ Optional | Food, Water, Medical, etc.       |
| Photo           | ❌ Optional | Photo of the situation           |

4. Click **"Submit Report"**

**Severity Guide:**

| Level       | When to Use                                   |
| ----------- | --------------------------------------------- |
| 🟢 Low      | Minor incident, no immediate danger           |
| 🟡 Medium   | Significant impact, help needed               |
| 🟠 High     | Serious situation, urgent response needed     |
| 🔴 Critical | Life-threatening, emergency response required |

### 3. Report Animal Rescue Need

**Purpose:** Report an animal that needs rescue.

**Steps:**

1. Navigate to **"Animal Rescue"**
2. Click **"Report Animal"**
3. Fill in the form:

| Field         | Required    | Description                       |
| ------------- | ----------- | --------------------------------- |
| Animal Type   | ✅ Yes      | Dog, Cat, Bird, Livestock, etc.   |
| Breed         | ❌ Optional | If known                          |
| Condition     | ✅ Yes      | Injured, Trapped, Stranded, etc.  |
| Is Dangerous? | ✅ Yes      | Check if animal may be aggressive |
| Description   | ✅ Yes      | Animal's condition, situation     |
| Location      | ✅ Yes      | Where the animal is               |
| Photo         | ❌ Optional | Photo of the animal               |

4. Click **"Submit Report"**

> ⚠️ **Safety First:** If the animal appears dangerous, mark "Is Dangerous" so responders can prepare appropriately.

### 4. Request a Relief Camp

**Purpose:** Request registration of a new relief camp location.

**Steps:**

1. Navigate to **"Camps"** from the responder interface
2. Click **"Request New Camp"**
3. Fill in camp details:

   - Camp Name
   - Type (Temporary Shelter, Medical Facility, etc.)
   - Location
   - Capacity
   - Available Facilities
   - Contact Information

4. Submit for admin approval

> 📝 **Note:** Camp requests must be approved by an administrator before appearing on the public list.

### Using the Location Picker

When selecting a location:

1. **Click on the Map** - Click anywhere on the map to set a pin
2. **Search Address** - Type an address in the search box
3. **Use Current Location** - Click the GPS button to use your device location

The map shows:

- 📍 Your selected location
- 🏕️ Nearby relief camps
- 📊 Disaster incident markers

## For Responders

As a responder, you can view reports, offer assistance, and contribute to relief efforts.

### 1. View and Respond to Reports

**Finding Reports:**

1. Click **"I Want to Help"** from the landing page
2. Choose a category:
   - Missing Persons List
   - Disaster Reports List
   - Animal Rescue Needs

**Viewing Details:**

1. Click on any report card to see full details
2. View location on the map
3. See contact information
4. Check current status

**Marking as Resolved:**

For Missing Persons:

1. Open the missing person detail
2. Click **"Mark as Found"**
3. Fill in:
   - Your contact information
   - Notes about how they were found
4. Submit

For Disasters:

1. Open the disaster detail
2. Click **"Mark as Resolved"**
3. Provide resolution details
4. Submit

### 2. Find Relief Camps

**Steps:**

1. Navigate to **"Camps"**
2. View all active camps on the map
3. Filter by:
   - District
   - Camp Type
   - Available Capacity

**Camp Information:**

- 📍 Location and address
- 👥 Capacity and current occupancy
- 🏥 Available facilities
- 📞 Contact information
- 📊 Needs (supplies required)

### 3. Register as a Volunteer

**Steps:**

1. Navigate to **"Volunteers"**
2. Click **"Register as Volunteer"**
3. Fill in your details:

   - Name and contact
   - Skills and availability
   - Preferred activities
   - Group information (if any)

4. Submit registration

### 4. Make a Donation

**Steps:**

1. Navigate to **"Donations"**
2. View the current donation total and goal
3. Click **"Donate Now"**
4. Select amount or enter custom amount
5. Fill in your details:

| Field   | Required    | Description              |
| ------- | ----------- | ------------------------ |
| Amount  | ✅ Yes      | Donation amount          |
| Name    | ❌ Optional | Your name (or anonymous) |
| Email   | ✅ Yes      | For receipt              |
| Message | ❌ Optional | Supportive message       |

6. Enter payment details (Stripe secure payment)
7. Complete donation

**Donation Features:**

- 💰 **Live Counter** - See total donations in real-time
- 📊 **Transparency** - View all donations in the public ledger
- 🎯 **Goals** - Track progress toward relief goals
- 📧 **Receipt** - Receive email confirmation

## For Administrators

Administrators have full access to manage the platform.

### Accessing Admin Panel

1. Navigate to **"/admin/login"**
2. Enter your credentials:
   - Email
   - Password
3. Click **"Login"**

> 🔐 **Security:** Admin credentials are managed through Supabase Auth. Contact the system administrator for access.

### Admin Dashboard

After login, you'll see the dashboard with:

- 📊 Statistics overview
- 📝 Pending requests
- ⚡ Recent activity
- 🔔 Alerts

### 1. Review Camp Requests

**Steps:**

1. Click **"Review Requests"** in the admin menu
2. View all pending camp requests
3. For each request:
   - Review camp details
   - Check location validity
   - Verify contact information
4. **Approve** or **Reject**:
   - Click **"Approve"** to add camp to public list
   - Click **"Reject"** with notes explaining why

### 2. Manage Camps

**Steps:**

1. Click **"Manage Camps"**
2. View all registered camps
3. Actions available:
   - **Edit** - Update camp information
   - **Update Occupancy** - Change current numbers
   - **Mark Closed** - Close a camp
   - **Delete** - Remove camp record

**Editing a Camp:**

1. Click **"Edit"** on a camp card
2. Update fields as needed
3. Click **"Save Changes"**

### 3. Register New Camp (Direct)

Admins can register camps directly without approval:

1. Click **"Register Camp"**
2. Fill in complete camp details
3. Click **"Register"**
4. Camp appears immediately on public list

### 4. View All Records

**Steps:**

1. Click **"All Records"**
2. Select category:
   - Missing Persons
   - Disasters
   - Animal Rescues
   - Camps
3. Use filters and search
4. Export data if needed

### 5. Analytics

View platform statistics:

- Total reports by category
- Resolution rates
- Active vs resolved cases
- Donation totals
- Camp occupancy rates

## FAQs

### General Questions

**Q: Do I need to create an account?**
A: No! Reporters and responders can use the platform without an account. Only administrators need to login.

**Q: Is my information secure?**
A: Yes. We use industry-standard security practices and your data is stored securely in Supabase.

**Q: Can I update a report after submission?**
A: Responders can update status. For major changes, submit a new report or contact an admin.

### Reporter Questions

**Q: What if I don't know the exact location?**
A: Provide the best estimate you can. You can describe landmarks or approximate areas.

**Q: How will I know if someone responds to my report?**
A: The status will change when responders take action. Check back on the platform for updates.

**Q: Can I report anonymously?**
A: You must provide contact information for coordination, but your details are not publicly visible.

### Responder Questions

**Q: How do I know which reports are most urgent?**
A: Reports are color-coded by severity. Red/Critical reports need immediate attention.

**Q: Can I coordinate with other responders?**
A: Yes, use the contact information provided in reports to coordinate.

**Q: How are donations used?**
A: All donations are tracked transparently. Check the donation ledger for details.

### Admin Questions

**Q: How do I add a new admin?**
A: Admin accounts are created in Supabase. Go to Authentication → Users to add new admins.

**Q: Can I export data?**
A: Yes, use the export function in the Records section or access data directly via Supabase.

**Q: What happens to rejected camp requests?**
A: They remain in the system with "Rejected" status. Users can submit new requests.
